# HTML, CSS & Bootstrap

**What it is:** HTML is the structure of a web page (headings, tables, links),
CSS is its appearance (colors, spacing, layout). **Bootstrap 5** is a
ready-made CSS library: you put its class names on your HTML and get a clean
look without writing much CSS yourself. This project uses Bootstrap via plain
class names — deliberately *without* the `react-bootstrap` wrapper library —
so the markup stays explicit.

## Next things to learn

- Basic HTML elements as they appear in our JSX: `<div>`, `<table>`, `<a>`,
